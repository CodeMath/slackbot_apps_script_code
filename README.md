# slackbot_apps_sciprt
> Slack Bot integrations with Google Apps
* Google Apps Script 코드 입니다. 

## json file

1. `blocks_kit.json`
* Slack 메시지 blocks 킷

2. `driveapp_api.json`
* 구글 드라이브 API

3. `library.json`
* 기타 라이브러리

4. `slack_sdk.json`
* Slack Bot API

위 4개를 GS에 등록해서 사용해야합니다.

`SlackEventSubs.gs` 를 이벤트 트리거로 등록하면 됩니다.
